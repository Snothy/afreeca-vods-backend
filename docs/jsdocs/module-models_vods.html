(static) createVodObject(bj_id, vods, cookie, addToDb) → {Array.<object>}

Given an array of vod objects, this method will retrieve their corresponding m3u8 playlist data. The playlist data is what makes up the content of the vod.

(static) fetchNewVod(bj_id, cookie) → {number}

This method was developed mostly for testing & learning purposes. The response will take a long time. Only used for streamers who are currently Live. It works by listening for new vods by saving the ID of the last available VOD for a specific streamer and then making requests until a new vod is found. It will periodically check whether the streamer is still Live, and if no vod was found for whatever reason, the fetching will stop.
Parameters:
Name Type Description
bj_id string A streamers unique identifier.
cookie string A cookie to identify the user making the request. The data for some vods can only be obtained if you are logged in.
Source:
  • models/vods.js, line 97
Returns:
1 on success and 0 on fail or error. The vod is saved to the database on success.
